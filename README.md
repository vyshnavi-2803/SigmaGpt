# SigmaGPT

SigmaGPT is a MERN Stack AI Chat Application that allows users to interact with an AI assistant powered by OpenAI's GPT-4o Mini API.

The application processes user questions, generates intelligent responses, and stores relevant conversation data in MongoDB.

## Tech Stack

Frontend
- React.js
- CSS
- Axios

Backend
- Node.js
- Express.js

Database
- MongoDB

AI Integration
- OpenAI GPT-4o Mini API

---

## Features

- AI powered chat interface
- Real time response generation
- Chat history storage in MongoDB
- REST API based backend
- Clean UI for conversation interaction
